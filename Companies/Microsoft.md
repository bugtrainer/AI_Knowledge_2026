# 🟦 Microsoft

## Overview
The platform king of AI. Made the decisive $13B investment in [[OpenAI]] and embedded AI across every Microsoft product. Azure is the #1 enterprise cloud for AI workloads. GitHub Copilot is the most widely-used AI coding tool. Under [[Satya Nadella]]'s leadership, Microsoft became the world's most valuable company.

## Flagship Products
- **Azure OpenAI Service** — Enterprise access to GPT, DALL-E, Whisper
- **[[Microsoft 365 Copilot]]** — AI in Word, Excel, PowerPoint, Outlook, Teams
- **[[GitHub Copilot _ Copilot Enterprise (Microsoft)]]** — AI pair-programming
- **[[Microsoft Copilot]]** — Free AI assistant across Windows, Edge, and Bing
- **Phi-4** — Small but powerful open model
- **[[AutoGen]] / Semantic Kernel** — AI agent frameworks

## Key Facts (2026)
- **$13B invested in [[OpenAI]]** — Largest single AI investment
- AI writes 30% of Microsoft's own code
- GitHub Copilot: most used AI coding tool globally
- Azure AI revenue growing 50%+ year-over-year
- Microsoft 365 Copilot becoming standard enterprise productivity tool
