## Definition
An **AI Agent** is an AI system that doesn't just answer questions — it autonomously takes a **sequence of actions** to complete a goal. It can use tools, browse the web, write code, send emails, query databases, and make decisions along the way.

## Real-World Analogy
A traditional AI assistant is like a knowledgeable colleague you ask questions. An AI agent is like giving that colleague a computer, access to all your systems, and asking them to **run an entire project end-to-end** — while you go do something else.

## How Agents Work
```
User Goal → Agent Plans → Agent Acts → Agent Observes → Agent Adjusts → Result
```
1. **Planning**: The agent breaks the goal into sub-tasks
2. **Tool Use**: It calls APIs, runs code, searches the web ([[Function Calling]])
3. **Observation**: It reads the results of its actions
4. **Reflection**: It evaluates whether it's on track ([[Reflection]])
5. **Iteration**: It adjusts its plan and continues ([[ReAct]] pattern)
