# 🦙 AnythingLLM

## Definition
Open-source desktop/web application to easily build private [[RAG]] pipelines and offline [[AI Agent]]s locally. Connect any [[LLM]] (local via [[Ollama]] or cloud API) to your documents, web pages, or databases — without coding. 20,000+ GitHub stars.

## How It Works (Simplified)
1. Install the desktop app (Mac, Windows, Linux) or run via [[Docker]]
2. Connect an LLM: local ([[Ollama]]) or API ([[OpenAI]], [[Anthropic]], etc.)
3. Upload documents (PDF, Word, web pages, code, Obsidian vaults)
4. AnythingLLM chunks, embeds, and stores documents in a local [[Vector Database]]
5. Ask questions grounded in YOUR data — fully private, fully offline

## Why It Matters in 2026
- Makes [[RAG]] accessible to non-programmers — no Python or coding required
- Ideal for professionals who want AI help on private documents (contracts, code, research)
- Combined with [[Ollama]], creates a **fully offline AI research assistant** on any laptop

> [!tip] Key Fact
> AnythingLLM reached 20,000 GitHub stars within weeks of launch — a record for open-source AI tools. It proved massive demand exists for private, code-free RAG solutions.
