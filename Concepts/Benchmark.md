# 📏 Benchmark

## Definition
A **Benchmark** is a standardised test used to compare AI model performance. Benchmarks provide objective scores across capabilities like reasoning, coding, math, and knowledge.

## Key Benchmarks in 2026
| Benchmark | What It Tests |
|---|---|
| **MMLU** | Massive Multitask Language Understanding (broad knowledge) |
| **ARC** | Abstract reasoning (AI2 Reasoning Challenge) |
| **HumanEval** | Code generation accuracy |
| **MATH** | Mathematical problem solving |
| **GPQA** | Graduate-level scientific reasoning |
| **SWE-Bench** | Real-world software engineering tasks |
| **Arena (LMSYS)** | Human preference voting (Chatbot Arena) |

## Why It Matters
- Benchmarks drive the "model race" — labs compete for top scores
- However, benchmark scores don't always reflect real-world usefulness
- "Benchmark gaming" (training on test data) is a known problem
- Human preference rankings (Arena) are increasingly valued over automated scores
