# 🖼️ CNN

## Definition
A **Convolutional Neural Network (CNN)** is a [[Neural Network]] architecture specialised for processing **grid-like data** — especially images. It uses "convolutional filters" that slide across the image to detect features like edges, textures, and shapes.

## Why It Matters
- Pioneered by [[Yann LeCun]] in the 1990s
- Basis for all image recognition, object detection, and computer vision
- Used in self-driving cars, medical imaging, facial recognition
- While [[Transformer]]s are now also used for vision, CNNs remain important

## Key Relationships
- Pioneered by: [[Yann LeCun]]
- Type of: [[Neural Network]], [[Deep Learning]]
- Used for: Computer vision, image recognition
- Complemented by: [[Transformer]] (Vision Transformers)
