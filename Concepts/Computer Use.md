# 🖱️ Computer Use

## Definition
**Computer Use** is an AI model's ability to interact with software environments the way a human does — clicking buttons, typing text, navigating desktop applications, filling forms, and reading screens.

## Why It Matters in 2026
- [[Claude (Anthropic)|Claude]] and [[ChatGPT (OpenAI)|GPT-5.4]] both support computer use natively
- Enables [[AI Agent]]s to work with ANY software — even legacy apps with no API
- [[Perplexity Computer]] runs AI agents on your desktop 24/7
- [[Comet (Perplexity)]] browses the web autonomously
