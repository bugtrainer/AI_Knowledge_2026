## Definition
A **Foundation Model** is a large [[Pre-Training|pre-trained]] AI model that is adaptable to many downstream tasks. Instead of training a separate model for each application, you start with a single powerful foundation and [[Fine-Tuning|fine-tune]] or prompt it for specific needs.

## Examples in 2026
- GPT-5.4, Claude Opus 4.6, Gemini 3.1, Llama 4 — all are foundation models for text
- Stable Diffusion 4 — foundation model for image generation
- Whisper (OpenAI) — foundation model for speech recognition

## Key Relationships
- Trained via: [[Pre-Training]]
- Adapted via: [[Fine-Tuning]], [[LoRA]], [[Prompt Engineering]]
- Type of: [[Deep Learning]] model
- Related: [[LLM]], [[Generative AI]]
