## Definition
The **Model Context Protocol (MCP)** is an open standard created by [[Anthropic]] for connecting AI models to external tools, APIs, and data sources. Think of it as a **universal plug standard** that lets any AI assistant use any tool — without custom integrations for each one.

## Real-World Analogy
Before USB, every device had its own proprietary connector. MCP is the "USB-C for AI" — a single standard that lets any AI model talk to any external service: databases, file systems, APIs, code editors, messaging apps.

## Why It Matters in 2026
MCP has become the **dominant standard** for agent tool use:
- Supported by: **Google, Microsoft, NVIDIA**, and hundreds of integrations
- Every major coding IDE (Cursor, Windsurf, Antigravity) supports MCP
- [[OpenClaw]] and [[NemoClaw (NVIDIA)]] use MCP as the core tool connection layer
- Essential infrastructure for the [[AI Agent]] revolution
