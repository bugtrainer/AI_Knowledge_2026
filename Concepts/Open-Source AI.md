# 🌐 Open-Source AI

## Definition
AI models released with **open weights** — downloadable and self-hostable by anyone. Llama ([[Meta AI]]), Mistral, Qwen, Phi, and DeepSeek are key examples. The open-source movement enables privacy, customisation, offline use, and scientific reproducibility. Debate: "open weights" ≠ "fully open" since training data and code are often not released.

## How It Works (Simplified)
1. A lab trains a frontier model (investing millions in compute)
2. Instead of only offering an API, they release the trained weights as downloadable files
3. Anyone can download, run locally ([[Ollama]]), fine-tune ([[LoRA]]), and modify the model
4. Community creates thousands of derivative models hosted on [[Hugging Face]]

## Why It Matters in 2026
- **500,000+ models** now on Hugging Face Hub — the vast majority open-source
- [[Meta AI]] made a strategic bet that open-sourcing Llama would commoditise the market and hurt API-only competitors
- Open-source models enable running AI **fully offline** and **privately** — crucial for healthcare, military, finance
- [[DeepSeek]] R1 (open-source) matching closed-source o1 was a watershed moment proving open models can compete

> [!tip] Key Fact
> Llama 2's open-source release (2023) spawned 10,000+ derivative models in 3 months — the "Linux moment" for AI.
