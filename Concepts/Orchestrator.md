# 🎼 Orchestrator

## Definition
An **Orchestrator** is a controller that coordinates and delegates work to multiple [[AI Agent]]s — deciding which agent handles which sub-task, managing handoffs, and synthesising results. It's the "manager" in a multi-agent system.

## Why It Matters in 2026
- As AI systems grow more complex, single agents aren't enough
- [[CrewAI]] — Role-based multi-agent orchestration
- [[LangChain _ LangGraph]] — Graph-based agent workflows with checkpointing
- [[OpenClaw]] — Orchestrates local AI agents for personal tasks
