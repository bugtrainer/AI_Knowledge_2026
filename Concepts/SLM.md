## Definition
A **Small Language Model (SLM)** is a lightweight, efficient AI model (typically <10B parameters) designed to run **locally on devices** — phones, laptops, IoT hardware — without cloud connectivity.

## Key SLMs in 2026
- **Gemma 4** (Google) — Open-source, on-device
- **Phi-4** (Microsoft) — Punches above its weight class
- **Mistral Ministral** — Edge-optimised
- **Llama 4 (small variants)** (Meta) — Mobile-ready

## Why It Matters
- Privacy: Data never leaves your device
- Speed: No network latency
- Cost: No API fees
- Offline: Works without internet
- Combines with [[Quantisation]] for even smaller footprints

## Key Relationships
- Deployed on: [[Edge AI]], [[NPU]], [[ESP32]], [[Jetson Nano]]
- Compressed via: [[Quantisation]], [[Distillation]]
- Tools: [[Ollama]]
- Contrast with: [[LLM]] (large cloud-based models)
