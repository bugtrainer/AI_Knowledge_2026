## Definition
**Vibe Coding** is using AI to write most or all of the code in a software project — by describing what you want in **plain English**. Coined by **Andrej Karpathy** (former OpenAI research lead), it's now mainstream in 2026.

## Scale of Adoption in 2026
- AI writes ~30% of Microsoft's code
- AI writes >25% of Google's code
- Non-engineers are building functional apps with [[Lovable]] and [[Replit Agent]] in hours
- [[Devin 2.0 (Cognition)]] provides a fully autonomous AI software engineer for $20/month

## Key Tools for Vibe Coding
- [[Cursor]] — Most popular AI IDE
- [[Claude Code (Anthropic)]] — Terminal-based coding agent
- [[GitHub Copilot _ Copilot Enterprise (Microsoft)]] — Most widely adopted in enterprises
- [[Google Antigravity]] — Agent-first IDE by Google
- [[Lovable]] — Build full-stack apps through conversation
- [[Bolt.new (StackBlitz)]] — Browser-based, zero local setup
