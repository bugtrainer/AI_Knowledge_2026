# AI Knowledge Encyclopedia (2026)

A comprehensive, interlinked knowledge base of the Artificial Intelligence landscape, designed to be viewed as an interactive graph in [Obsidian](https://obsidian.md/).

## 🌟 What This Is
This repository contains a full encyclopedia of the modern AI era. It explores frontier concepts, key people, major companies, and bleeding-edge tools. Every file is written in Markdown and heavily cross-linked. 

## 🚀 How to Use This
This is an **Obsidian Vault**. You can read the Markdown files natively on GitHub, but to get the best experience (including the interactive knowledge graph):

1. Clone or download this repository to your computer.
2. Download and install [Obsidian](https://obsidian.md/) (it's free).
3. Open Obsidian and click **"Open folder as vault"**.
4. Select the downloaded folder.
5. Hit `Ctrl/Cmd + G` to open the Graph View and explore the connections! 

## 🗺️ Navigation
Start your journey at the `_Index.md` file, which serves as the master dashboard, or explore one of the hub Maps of Content (MOCs):
- `MOC - Core Concepts.md`
- `MOC - Training & Optimisation.md`
- `MOC - Agents & Autonomy.md`
- `MOC - Safety & Alignment.md`
- `MOC - Infrastructure & Deployment.md`

## 📊 Features
- **250+ Interlinked Notes:** Concepts, Tools, Companies, People, and a 110+ term Glossary.
- **Mastery Tracking:** Each note has YAML frontmatter with a `mastery_score` (0-5) to track your learning.
- **Study Dashboard:** Use the `Study Dashboard.md` to track your progress from Beginner to Advanced topics.
- **Zero Lock-in:** Everything is written in standard Markdown.
