# 🟢 ChatGPT (OpenAI)

## What It Does
[[OpenAI]]'s conversational AI assistant — the app that ignited the generative AI revolution. Reached 100M users in 2 months (Nov 2022), the fastest-growing consumer application in history. Powers coding, writing, research, image generation, and now web browsing and computer use.

## Key Features
- **GPT-5.4** — Flagship model with 1M-token context, multimodal, computer use
- **o3/o3-pro** — [[Reasoning Model]]s that "think" before answering (best for math, code, science)
- **DALL-E 4** — Integrated image generation
- **Sora 2** — Video generation capability
- **ChatGPT Search** — Real-time web browsing with citations
- **Computer Use** — Can control mouse, keyboard, and screen

## Who It's For
Everyone — from casual users asking questions to developers building on the GPT API. The most widely-used AI product globally with hundreds of millions of users.
