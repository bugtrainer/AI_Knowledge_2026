# 🧠 Claude (Anthropic)

## What It Does
[[Anthropic]]'s AI assistant, regarded as the most reliable and thoughtful LLM for coding, writing, analysis, and enterprise use. Available at claude.ai and via API. Known for safety-first design ([[Constitutional AI]]) and industry-leading context windows.

## Key Features
- **Claude Opus 4.6** — Most capable model: 1M-token context, 14.5-hour autonomous task window
- **Claude Sonnet 4.6** — Balanced model: fast, smart, default for most users
- **Claude Haiku 4.5** — Fastest/cheapest: ideal for high-volume API usage
- **Claude Code** — CLI tool for autonomous coding sessions up to 14.5 hours
- **[[MCP]] Integration** — First major LLM to adopt universal tool connections

## Who It's For
Developers, researchers, writers, and enterprises needing reliable AI that excels at long-context reasoning, code generation, and careful instructions-following.
