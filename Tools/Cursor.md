# ⌨️ Cursor

## What It Does
AI-native code editor (fork of VS Code) that uses [[LLM]]s to code alongside you. Tab-complete, whole-file edits, multi-file refactoring, and "Composer" mode for building features from natural language descriptions. The most popular AI coding tool for full-stack development.

## Key Features
- **Composer** — Multi-file AI editing using natural language instructions
- **Tab Completion** — Context-aware autocomplete using your entire codebase
- **Chat** — Ask questions about your code, get explanations and fixes
- **Multi-Model** — Choose from Claude, GPT-4, Gemini as your AI backend
- **Codebase-Aware** — Indexes your entire project for context-rich suggestions

## Who It's For
Software developers who want AI pair-programming without leaving their editor. From beginners doing [[Vibe Coding]] to senior engineers using it for rapid prototyping.
