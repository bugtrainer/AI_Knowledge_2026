# 💎 Gemini (Google)

## What It Does
[[Google DeepMind]]'s flagship AI model family. Natively multimodal — processes text, images, audio, and video from the ground up (not bolted on). Integrated across Google Search, Workspace, Android, and Cloud. Deep Think mode for advanced reasoning.

## Key Features
- **Gemini 3.1 Pro** — Flagship model with 1M-token context and Deep Think mode
- **Gemma 4** — Open-source small models for local/edge use
- **Google AI Mode** — AI-powered search results in Google Search
- **Workspace Integration** — Built into Gmail, Docs, Sheets, Slides
- **Android AI** — On-device AI features across Android devices

## Who It's For
Google ecosystem users, developers using Google Cloud AI, and anyone wanting a multimodal AI assistant deeply integrated with the world's most used software.
