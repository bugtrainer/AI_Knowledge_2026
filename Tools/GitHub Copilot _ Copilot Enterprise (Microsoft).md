# 🐙 GitHub Copilot (Microsoft)

## What It Does
[[Microsoft]]'s AI pair-programming tool integrated into VS Code, JetBrains, and Neovim. The most widely deployed AI coding tool globally. Copilot Enterprise adds codebase-aware chat, PR summaries, and internal knowledge base search.

## Key Features
- **Inline Completion** — Real-time code suggestions as you type
- **Copilot Chat** — Conversational coding assistant in the editor
- **Copilot Enterprise** — Organization-wide codebase awareness and PR automation
- **Multi-Model** — Uses GPT-4 and Claude under the hood
- **Workspace Mode** — Understands your entire repository structure

## Who It's For
Software developers at all levels using VS Code or JetBrains IDEs. Enterprise teams wanting organisation-wide AI coding assistance.
