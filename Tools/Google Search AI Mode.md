# 🔍 Google Search AI Mode

### What It Is / Does
Google's AI-powered search mode, integrating [[Gemini (Google)]] directly into search results. Provides synthesised answers alongside traditional links. Competes with [[Perplexity AI]] and [[ChatGPT Search]].

### Key Connections
- Made by: [[Google DeepMind]] ([[Sundar Pichai]])
- Uses: [[Gemini (Google)]], [[RAG]], [[Grounding]]
- Related: [[GEO]] (Generative Engine Optimisation)
