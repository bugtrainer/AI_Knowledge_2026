# 🚀 Grok (xAI)

## What It Does
[[xAI]]'s AI assistant, built by [[Elon Musk]]'s AI company. Known for real-time access to X (Twitter) data, longer context windows than competitors (2M tokens), and fewer content restrictions. Uses a multi-agent architecture for cross-verification.

## Key Features
- **Grok 4.20 Beta 2** — Flagship model with 2M-token context
- **Multi-Agent Architecture** — Uses multiple models for cross-verification
- **Real-Time X Data** — Unique access to live social media data
- **Image Generation** — Built-in via Aurora model
- **Fewer Restrictions** — More permissive than ChatGPT/Claude on controversial topics

## Who It's For
X (Twitter) power users, researchers wanting less-filtered AI, and users needing massive context windows.
