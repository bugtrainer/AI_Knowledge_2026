# 🤗 Hugging Face

## What It Does
The "GitHub of AI" — the world's largest open-source AI platform hosting **500,000+ models**, 100K+ datasets, and 300K+ Spaces (demo apps). Every open-source AI model is published here. The `transformers` Python library is the standard way to load and use AI models.

## Key Features
- **Model Hub** — 500K+ downloadable pre-trained models
- **Datasets Hub** — 100K+ open datasets for training
- **Spaces** — Free GPU-powered demo hosting
- **Transformers Library** — Python library to load any model in 3 lines of code
- **Inference API** — Run any model via simple API calls

## Who It's For
AI researchers, developers, and hobbyists who want to use, share, or deploy open-source AI models.
