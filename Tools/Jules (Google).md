# Jules (Google)

### What It Is / Does
Google's asynchronous coding agent. You assign Jules a task, it clones your repository into a cloud VM, works on it asynchronously (you can close your computer), and returns a pull request. Powered by Gemini 3 Pro. Jules V2 (codenamed 'Jitro') in development — expected at Google I/O 2026.
