# 💜 Lovable

## What It Does
AI-powered full-stack web application builder. Describe what you want in plain English and Lovable generates a complete, deployable web app with React frontend, Supabase backend, and authentication. The exemplar of [[Vibe Coding]].

## Key Features
- **Natural Language → Full App** — Describe features, get a running app
- **React + Supabase** — Generates production-quality full-stack code
- **Real-Time Preview** — See your app as it's being built
- **One-Click Deploy** — Deploy to production instantly
- **Iterative Editing** — Chat to refine and modify features

## Who It's For
Non-developers who want to build web apps, and developers who want rapid prototyping.
