# 🔌 MCP (Model Context Protocol)

## What It Does
[[Anthropic]]'s open standard for connecting AI models to external tools, databases, and APIs. Like "USB-C for AI" — one universal interface instead of custom integrations for every tool. Adopted by [[Claude (Anthropic)]], [[Cursor]], and many AI platforms.

## Key Features
- **Universal Standard** — One protocol works with any compliant AI model
- **Server Architecture** — Tools expose capabilities as MCP servers
- **Resource Access** — Models can query databases, read files, call APIs
- **Tool Discovery** — AI automatically discovers and uses available tools
- **Security Model** — Controlled permissions for what tools can access

## Who It's For
Developers building AI agents that need to interact with external systems. Teams wanting to create reusable tool connections across multiple AI platforms.
