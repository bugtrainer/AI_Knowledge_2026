# OpenClaw

### What It Is / Does
The most viral open-source project of 2026 — 250,000+ GitHub stars, fastest-growing open-source project in history. A personal AI agent running locally, connecting to messaging apps (WhatsApp, Telegram, Slack, Discord), and autonomously completing tasks 24/7.
